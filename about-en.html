Nyanko Studios

About Nyanko Studios

Nyanko Studios is an independent software development team that develops games and programs for multiple systems, including Windows and the Nintendo Entertainment System.

Current Projects

Our current Nintendo Entertainment System music project is a re-creation of the introduction sequence of the game UNDERTALE.

Our current video game project is a port of the mini-game "Biplanes" from "Triple Action" for the Intellivision to the Nintendo Entertainment System.

We are simultaneously working hard on moddifying Norix's Nintendo Entertainment System emulator (VirtuaNES) to improve the accuracy and add new features.

Our current project is a port of the classic Windows game Minesweeper to the Nintendo Entertainment System.
