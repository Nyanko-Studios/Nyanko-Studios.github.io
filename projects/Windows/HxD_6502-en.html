PROGRAM NAME

Platform: Windows
Version: 2.1
Release Date: 2020-06-30

This is a plugin for the program HxD. HxD is a fast hex-editor program, with the ability to quickly open and edit files well over 8192 GiB, RAM, and disk-drives. This plugin adds 6502 assembly support to HxD's "data inspector," with the ability to edit. Selecting "Show integers in hexadecimal base" enables the showing of "illegal" or "undocumented" opcodes. At this time, the plugin is compatible only with the 64-bit version of HxD.
